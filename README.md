# 🕉️ Bhagavad Gita AI Chatbot

**Divine Wisdom • Multi-Language Support**

A smart AI chatbot that helps you explore the **Bhagavad Gita** in English, Hindi, and Gujarati.

### ✨ Features
- Semantic search in any language
- Direct verse lookup (e.g. `1.4`, `18.66`)
- Gujarati meaning support
- Word-by-word meaning + Commentary
- Fast and responsive
